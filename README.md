# new-project

Scaffolded Node + TypeScript starter.

## Quickstart

Install dev deps and build:

```bash
npm install
npm run build
node dist/index.js
```

## Run (background)

Start the compiled server in background (uses port 4001):

```bash
npm run start-bg
```

Check health:

```bash
curl http://127.0.0.1:4001/health
```

## Tests

Run the built-in Node test runner:

```bash
npm test
```
